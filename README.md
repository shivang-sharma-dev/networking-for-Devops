## 📖 What is This?

This repository is a **self-paced learning guide** for DevOps engineers who want to deeply understand networking — not just the theory, but how it applies in real cloud and container environments.

Whether you're preparing for a certification, an interview, or just want to stop googling what a subnet mask is — this repo has you covered.

## 🚀 How to Use This Repo

1. Start with [`ROADMAP.md`](./ROADMAP.md) to understand the learning path
2. Go through notes in order inside `01-notes/`
3. Practice commands in `02-cheatsheet/`
4. Reinforce knowledge with `03-hands-on-labs/`
5. Build real projects in `04-projects/`
6. Prep for interviews using `05-interview-prep/`
