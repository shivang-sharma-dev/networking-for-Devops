# Contributing

Found a mistake? Want to add a diagram, lab, or resource?

1. Fork the repo
2. Create a branch: `fix/typo-in-dns` or `feat/add-bgp-notes`
3. Make changes
4. Commit: `fix: correct DNS TTL example in note 04`
5. Open a Pull Request

Guidelines:
- Use ASCII diagrams for clarity
- Include analogies where helpful
- Test all commands before submitting
- One topic per PR
